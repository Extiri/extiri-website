Features

Manage your snippets

Store code, text, links, color palettes, images, and prompts. Organize with groups, tags, and spaces. Visualize relationships with intuitive graph views for better data organization.

Document your snippets better

Use Markdown and LaTeX in descriptions. Add flow diagrams for visual documentation. AI can help generate titles and descriptions, streamlining your developer workflow.

Reuse your snippets

Flexible placeholders make snippets reusable—swap in your text, shell command results, or current date/time to fit any context.

Access snippets quicker

Side window, Snippets Store, AI chat, and Suggestions app—search and use snippets from your editor or with abbreviations. Semantic search makes finding the right snippet effortless.

Use snippets in other ways

Run shell scripts, preview HTML/Markdown, and get color values in multiple formats. JavaScript is supported by default, and you can add nearly any other language (see FAQ).

Online database of snippets

Snippets Store: browse, share, and add public snippets to your library. Keep your team's snippets organized in one place.

Useful tools in one place

Editor for code, Markdown, and HTML, RegEx tester, converters, generators, coders, and more—everything you need for efficient snippet management.

Integrated AI

Semantic search, AI-generated documentation, text/code extraction from images, and chatbot integration. CodeMenu works great alongside your other AI tools like GitHub Copilot. Learn more at our blog: 10 Reason To Use a Snippet Manager in the AI Era.

At this moment Editor supports JavaScript by default. But you can add any other language that has a CLI utility.

You can do this by placing language's command line utility in Application Scripts of CodeMenu (to open click "Open Application Scripts" in Languages settings section).

You can also do this by creating a symlink (shortcut to file, but accessible by CodeMenu) to command line utility by opening your terminal and then entering

cd /Users/(your username)/Library/Application\ Scripts/id.thedev.marcin.CodeMenu

and then

ln -s (path to utilities shell command) .

You can use which shell command to find the path. You might need to restart CodeMenu for language to show up.

To use LaTeX you just need to wrap an expression in $$ (for example: "Here is some fraction $$\frac{126}{3}$$"). Once in preview mode (eg. not editing) it will render on a separate line.

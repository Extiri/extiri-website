Apps that will help you create your apps




Apps

CodeMenu (macOS)

An advanced snippets manager. Store code (60+ languages), colors, RegEx and Markdown. Made to assist you at every point of a snippet's life. It has many features like spaces, stashes, code image recognition, semantic search, integrated AI, placeholders and MUCH more.

Suggestions (macOS)

An open source app that allows you to see "suggestions" directly in your IDE/editor. Currently it only supports CodeMenu. With CodeMenu provider, Suggestions allows you to search and use snippets and also use abbreviations.

Snippets Store (web)

An online database of code snippets. It is open source and can be self-hosted.

natsearch (cli, macOS)

A simple open source command line utility that allows you to search through text files (including code) using natural language.

CodeMenu

MacOS app

Learn more

An advanced snippet manager and organizer designed to support every stage of your snippet’s journey. Effortlessly store and manage code (in 55 languages), color palettes, links, prompts, text, and images—all in one elegant tool. Featuring powerful capabilities like visual graphs, customizable spaces and stashes, AI-powered semantic search, code image recognition, smart placeholders, and more, CodeMenu helps you work smarter.

Suggestions

MacOS app (open source)

Download
GitHub repo

                    

                    
An open source app that allows you to see list of suggestions directly in your IDE, code or text editor. You can either search or use automatic text replacement. Currently it only supports CodeMenu. With CodeMenu provider, Suggestions allows you to search and use snippets.
